# Meu Site Pessoal

Este repositório contém os arquivos necessários para o meu site pessoal, onde compartilho meus projetos, experimentos de aprendizado e informações sobre mim.

## Descrição

Meu site pessoal é uma plataforma onde apresento meus projetos de desenvolvimento, incluindo aplicativos, jogos, tutoriais e artigos relacionados à programação e tecnologia. Além disso, forneço informações sobre mim, meu background, interesses e contatos.

## Funcionalidades

- **Apresentação de Projetos:** Exibição de projetos de desenvolvimento, incluindo aplicativos web, jogos e experimentos.
- **Blog:** Seção dedicada a artigos, tutoriais e insights sobre programação, tecnologia e aprendizado.
- **Informações Pessoais:** Página com informações sobre mim, meu background, interesses e links para redes sociais.
- **Contato:** Formulário de contato para enviar mensagens diretamente para o meu e-mail.

## Tecnologias Utilizadas

- **HTML/CSS/JavaScript:** Linguagens de marcação e programação para desenvolvimento web.
- **Bootstrap:** Framework de CSS para design responsivo e componentes front-end.
- **GitHub Pages:** Plataforma de hospedagem gratuita para sites estáticos diretamente de repositórios do GitHub.

## Personalização

- **Adapte o Conteúdo:** Edite os arquivos HTML, CSS e JavaScript para adicionar seu próprio conteúdo e informações pessoais.
- **Estilo Visual:** Personalize o design e o estilo do site ajustando as classes e estilos CSS existentes.
- **Integração de Serviços:** Adicione integrações com serviços de terceiros, como formulários de contato e APIs de redes sociais, se desejar.
